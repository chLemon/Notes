# 稀疏数组

1. 记录数组有几行几列，有多少个不同的值
2. 把具有不同值的元素的行、列和值记录在一个小规模的数组中【3×（值数+1）】

小规模数组长这样：

|      | row  | col  | val  |
| ---- | ---- | ---- | ---- |
| 0    | ...  |      |      |
| 1    |      |      |      |
| 2    |      |      |      |

## 思路

1. 遍历原始数组，得到有效数据的个数sum
2. 创建稀疏数组```sparseArr int[sum+1][3]```
3. 存数据

恢复逆过来就行
